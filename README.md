INSTALL POSTGRESQL ON YOUR MACHINE
CHECK IF PSOTGRESQL SERVICE IS STARTED: sudo systemctl status postgresql
START PSQL SERVICE IF NOT STARTED: sudo systemctl start postgresql
SET UP PSQL DATABASE WITH THIS COMMAND: sudo -u postgres psql -f src/db/mock/setup.sql
INSTALL ALL PACKAGES FOR APP: yarn install
SEED THE DATABASE WITH SCHEMA AND MOCK DATA WITH THIS COMMAND: yarn seed
START HMTM WITH THIS COMMAND: yarn dev
